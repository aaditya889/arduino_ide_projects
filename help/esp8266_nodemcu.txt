To start coding on NodeMCU/ESP8266:

1. Add this in in Arduino -> preferences -> Additional boards manager URL: "http://arduino.esp8266.com/stable/package_esp8266com_index.json"
2. Install the esp8266 board from tools -> boards -> boards manager -> search for esp8266, and install
3. Select the right board: tools -> Board -> NodeMCU 1.0 (ESP12-E module)
4. Baud rate: 115200, CPU frequency: 80MHz, Programmer: USBasp
